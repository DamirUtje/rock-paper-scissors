Game rules

In addition to the default set of allowed hand signs rock, paper and scissors, the game contains an extended set with the hand sign "Well". At the start of the game you can choose between the two modes "Classic" and "Extended".

"Classic" mode:

  • Rock beats Scissors
  • Scissors beats Paper
  • Paper beats Rock

    • "Extended" mode:

  • Rock beats Scissors
  • Scissors beats Paper
  • Paper beats Rock and Well
  • Well beats Rock and Scissors

    • The game is played in best-of mode. If a player wins the predefined odd number of rounds, the entire game is considered won. At the start, the number of desired game rounds can be selected.
